## Instructions

Make sure you follow the instructions #1, 2, 3 in `backend` folder before proceeding next steps

```bash
cd backend

npm install

npm run dev
```

```bash
cd frontend

npm install

npm run dev
```

Open the app at `http://localhost:5173`
